Kerjain merupakan sebuah skill bridging platform yang menyediakan roadmap education bagi siswa/i (student) pendidikan smk dan vokasi serta job listing untuk para student serta profesional. di dalam platform ini ada 3 role (student, professional, dan admin). semua user dapat login dengan google, ataupun hanya dengan email & password. setelah login/register, mereka harus melengkapi data diri (menyesuaikan jika mereka menggunakan google)
student : Nama lengkap, photo profile, nama sekolah, bidang/industri, nomor wa, alamat email, linkedin link, instagram link (opsional), youtube link (opsional), tiktok link (opsional), biodata/deskripsi diri.
professional : Nama lengkap, photo profile, nama perusahaan, bidang/industri, nomor wa, alamat email, linkedin link, social media link lainnya (opsional)
Admin : hanya untuk admin utama apps ini -> fitur admin adalah memantau total user (all role, by role), management user (crud user), melihat total job listing, me-nonaktifkan job listing yang ada, melihat informasi user.

kemudian masuk kedalam fitur;
student : management profile, melihat semua job listing, filter job list by industries, total applicant (least and most), by match score, by type (internship, part time, full time, contract, wfo, wfh, hybrid), dan yang sudah di apply, daily streak (seperti tiktok, dan platform lainnya yang mempunyai streak), apply job (single apply button, dan di sisi profesional aka mendapatkan informasi detail applicant seperti nama, deskripsi diri, match score, kemudian dapat view profile applicant, alamat email dan whatsapp), AI Bases Assesment (student memiliki 2 fitur assessment; 1. Interest assessment pertanyaan dari AI untuk menentukan interest field mereka di mana dan level mereka sudah sejauh apa terkait field tersebut, kemudian generate roadmap tree untuk skill yang harus mereka pelajari (seperti roadmap.sh) jadi akan di generate tree node/perpoint untuk roadmap tree nya, 2. Self Assessment & Knowledge check untuk mengetahui update mereka terkait roadmap yang mereka pelajari. use flow dari Self assessment adalah, user akan memilih salah satu tree node yang ada pada roadmap, kemudian ai akan memberikan pertanyaan, dan penilaian berupa feedback singkat, menghitung total benar, dan apakah lulus atau tidak (minimal lulus 85%) ketika lulus, maka bisa lanjut belajar ke node berikutnya, jika belum maka tetap di situ dan user dapat kembali mengambil self assesment 5 menit kemudian. lalu fitur progress tracker, crud portfolio (seperti linkedin post namun field yang perlu di isi adalah maksimal 2 gambar, link external, dan deskripsi singkat) di profesional ketika melihat user profile applicant akan menampilkan porfolio, jadi user bisa pada dashboard bisa menentukan apakah portfolio nya public atau private.

profesional : management profile, management job (status : open/closed; title, deskripsi, type: internship/part time/full time/contract/daily worker; site:wfo, wfh, hybrid, melihat total applicant, melihat list applicant all dan spesfik job, detail applicantm melihat profile applicant, required skills (input sendiri berubah text akan menjadi array dan digunakan untuk matching score system)

admin : view total user, total lowongan, total lowongan aktif, total lamaran, total user, job listing, crud semuanya dan fitur yang sudah di sebutkan di atas

setiap user role akan memiliki dashboard masing-masing jadi endpoint nya dan response nya pun harus di perhatikan. jangan ragu untuk membuat sebuah endpoint yang besar, pastikan juga code quality nya bagus, error handling aman (minimal 3 cases : success, failed, unnormal cases seperti input salah), rate limiting, API Versioning, dan validation.
